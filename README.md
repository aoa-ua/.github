# AOA · `.github`

This repository contains the **public organization profile** and shared GitHub
configuration (issue templates, funding info, contributor docs) for the
[AOA organization](https://github.com/aoa-ua).

## What's here

- [`profile/README.md`](./profile/README.md) — rendered as the organization's
  public profile at https://github.com/aoa-ua
- _(future)_ shared issue / PR templates, `FUNDING.yml`, contributor guidelines

## About AOA

AOA is an ecosystem of in-person meetings, communities, and events.
NFC-verified venue rooms, daily AOA Match meetups, no swipes, no chats.

→ **Website**: https://aoa.com.ua
→ **For venues**: https://aoa.com.ua/for-business
→ **For organizers**: https://aoa.com.ua/for-organizers
→ **Help center**: https://aoa.com.ua/help
